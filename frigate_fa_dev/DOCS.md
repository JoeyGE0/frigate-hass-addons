This add-on tracks **dev branch** builds of Frigate (commit-pinned Docker images from GHCR). These are bleeding-edge and may be unstable — back up your config and database before updating.

You must create a config file in your add-on configuration folder ([click here to learn more](https://docs.frigate.video/configuration/#accessing-add-on-config-dir)).

Frigate brings realtime object detection to any camera video feed supported by FFmpeg. More detailed docs are maintained [here](https://docs.frigate.video).

## Required Dependencies

- MQTT: Frigate communicates with Home Assistant via MQTT

## Support

This is an unofficial dev add-on fork. For Frigate issues, see the [main repo](https://github.com/blakeblackshear/frigate/issues/new/choose).
